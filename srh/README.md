# Disky Chat Only (Static Site)

## What it is
A simple, nice-looking single page that embeds your Copilot Studio web chat in an iframe.

## Embed URL used
https://copilotstudio.microsoft.com/environments/Default-8492f3d5-34d6-4288-9225-2883ded976fb/bots/crd95_Disky/webchat?__version__=2

## Setup
- Unzip
- Open `index.html` in a browser

## Replace Disky image
Swap `assets/Disky.png` with your preferred image (same name).
